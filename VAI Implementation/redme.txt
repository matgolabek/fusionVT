To run the app:
1. Open docker
    sudo bash ./docker_run.sh xilinx/vitis-ai:latest
2. Select environment
    conda activate vitis-ai-pytorch
3. import nndct (IMPORTANT!)
    python -c "import pytorch_nndct"
4. Run Jupyter server
    jupyter notebook
5. Open browser clinking the link
6. For the first run uncomment !pip commands
